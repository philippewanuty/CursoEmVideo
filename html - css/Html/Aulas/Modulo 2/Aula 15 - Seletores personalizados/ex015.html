Seletores Personalizados

Identificando um elemento

Resumo:

Nesta aula aprendi sobre os seletores personalizados.
Quando você tem várias tags iguais voce pode identificar ela para que voce possa trata-la de maneira isolada.
Basta apenas você colocar o parametro id=" " e o nome que essa tag irá assumir.
No CSS você deve chamar a tag colocando um hash antes do novo nome da tag ex: #paragrafo1

Classificando um elemento

Resumo:

A classificação class=" " de um elemento é semelhante a funcionalidade da identificação, porém a grande diferença é com a classificação, voce pode usar em vários elementos, e a identificação apens um só.
No CSS você deve chamar as tags colocando um ponto antes do novo nome da tag ex: .paragrafo12
