Tag utilizadas

<picture> : utilizada para trabalhar com a imagem

<source> : utilizada para mudar de imagem dinamicamente

<audio> : utilizada para colocar um áudio no site

Parâmetros para audio:

  • controls: aparece o player
  • Autoplay: toca automaticamente
  • loop: toca infinito
  • preload=: formato de pré carregamento

<video> : utilizada para colocar um video no site

Parâmetros para video:

  • controls: aparece o player
  • Autoplay: toca automaticamente
  • Poster: foto de abertura
  • preload=: formato de pré carregamento
