Modelo de caixas

Nesta aula eu aprendi que tudo em html está dentro de uma caixa, aprendi sobre os termos de uma caixa como padding, margin, border,altura e largura. Aprendi também que no chrome existe o devtools, uma ferramenta que te auxilia na visualização e no teste dos códigos, esta ferramenta é uma mão na roda para realizar testes sem alterar o código.

Eu sou uma caixa

Termos utilizados
