PseudoElementos

Pseudo-elemento CSS é uma palavra-chave adicionada às declarações de um seletor após dois sinais de dois pontos e permitem que você formate um pedaço específico do elemento referenciado. Os principais pseudo-elementos usados nas CSS são ::before, ::after, ::first-letter, ::first-line.

Exemplos:

  • Google
  • Hotmail
  • GitHub

PseudoClasses usados

  • hover: efeito ativado ao passar o mouse em cima

  • visited: efeito ativado após clicar no link

  • active: efeito ativado quando o link se mantém pressionado

PseudoElementos usados

  • after: adiciona algo ao final do conteúdo
  • before: adiciona algo no começo do conteúdo

Sinais usados no estudo

  • # = id

  • . = class

  • : = pseudo-class

  • :: = pesudo-element

  • > = children
